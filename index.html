Tourist Trip Planner

A C++ and Python application that helps users plan customized travel routes between cities in Europe and the Middle East. It models cities and distances using graph data structures and applies Dijkstra’s algorithm for intelligent pathfinding. Users can choose transportation modes (car, train, airplane), set trip preferences (Nature, History, Culture, Adventure, Relaxation), and collaborate on shared travel plans. The system supports cost and fuel estimation, CSV‑based save/load functionality, and user feedback collection.

Cognitive Health Monitoring System

This Software Engineering project applies AI and Data Science to monitor cognitive health through social‑media text analysis. The system detects early linguistic indicators of dementia and Alzheimer’s using ClinicalBERT and Named Entity Recognition (NER) for anonymization. It integrates UML design principles and software architecture patterns to ensure scalability and maintainability. The project also includes Arabic textual analysis for multilingual data, enabling culturally adaptive AI models.
